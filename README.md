# Little Doors

ブランド／キャラクターコンサルティング Little Doors の公式サイト。

## 構成

- `index.html` — メインページ
- `images/` — 画像データ
  - 画像ファイル名はVercel/GitHubで安定して読み込めるように英数字へ統一済みです。

## 公開方法（Vercel）

1. このフォルダの中身をGitHubリポジトリのルートにアップロード
2. Vercelで該当リポジトリをImport
3. Framework Presetは `Other` のままDeploy
4. 以後、GitHubへ更新するとVercelに自動反映されます。
