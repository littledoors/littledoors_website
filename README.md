# Little Doors

ブランド／キャラクターコンサルティング Little Doors の公式サイト。

## 構成

- `index.html` — メインページ
- `images/` — イラスト・背景画像

## 公開方法

このリポジトリは GitHub Pages で公開できます。

1. リポジトリの **Settings → Pages** を開く
2. **Source** を `Deploy from a branch` に設定
3. **Branch** を `main`（または公開したいブランチ）、フォルダを `/ (root)` に設定
4. **Save** を押す

数分後、`https://<username>.github.io/<repo-name>/` で公開されます。
