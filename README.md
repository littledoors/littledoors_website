# Little Doors

ブランド／キャラクターコンサルティング Little Doors の公式サイト。

## ページ構成

- `index.html` — メインページ
- `services.html` — サービス詳細ページ
- `profile.html` — 代表プロフィール詳細ページ
- `images/` — イラスト・背景画像
- `支援領域イメージ図.png` — Services 内で使用する図

## 公開方法（GitHub Pages）

1. このフォルダの中身をリポジトリのルートにアップロード（フォルダごとではなく、ファイル単体を）
2. リポジトリの **Settings → Pages** を開く
3. **Source** を `Deploy from a branch` に設定
4. **Branch** を `main`、フォルダを `/ (root)` に設定
5. **Save** を押す

数分後、`https://<username>.github.io/<repo-name>/` で公開されます。
